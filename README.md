# Отчётогенератор 3000

## Что это за проект

Скрипт собирает данные из Excel-файлов **УПД\*.xlsx** (номер счёта-фактуры, грузополучателя и сумму), а затем формирует **отчёт Word** `report.docx`.

## Что лежит в проекте

- `main` — основной запуск (читает Excel, собирает данные, создаёт отчёт).
- `setup_logs` — настройка логов (куда и как пишутся сообщения).
- `utils` — вспомогательные функции для чтения Excel и поиска нужных полей.

## Как пользоваться (в 3 шага)

1. **Положите Excel-файлы** в папку `assets`
   - Файлы должны быть в формате **.xlsx** и начинаться с **`УПД`** (например: `УПД ... .xlsx`).

2. **Запустите скрипт**
   - Если проект запускается через Python:
     - откройте терминал в папке проекта
     - выполните команду:

       ```bash
       python main.py
       ```

     (файл `main` содержит точку входа `if __name__ == "__main__": ...`).

3. **Заберите результат**
   - Готовый отчёт появится в папке: `output/report.docx`

## Где смотреть лог (если что-то пошло не так)

- Актуальный лог: `logs/latest.log` (перезаписывается при каждом запуске)
- История запусков: `logs/runs/*.log` (хранятся последние ~30 запусков)
- Пример успешного лога — в `latest.log`.

## Если отчёт не сформировался

- Проверьте, что Excel-файлы лежат именно в `assets` и начинаются с `УПД`.
- Откройте `logs/latest.log` и посмотрите предупреждения вида “Не найдены якоря…” / “Не найдены значения…”.
